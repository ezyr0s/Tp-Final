Groupe 1 POEI Salesforce

Objectif de la formation

La préparation du stagiaire pour être capable de commencer sur le module Salesforce avec tous les acquis nécessaires, cela veut dire les bases de la programmation front avec du HTML, Css et Javascript ainsi que des outils , comme le git pour le versioning et Jira pour la méthode Agile..

Déroulement

La formation se déroule à distance via Teams, avec un formateur . les cours sont fragmentés en module après chaque module, des tps qui permettent de savoir le pourcentage de validation d'acquis du groupe .
